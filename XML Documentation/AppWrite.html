Class AppWrite
Provides functionality for rendering text onto a graphical surface.
Syntax
C#
    internal class AppWrite
Remarks
This helper class wraps the Graphics object and allows text to be drawn at a specified position with customizable font and color.

Method Write
Draws text onto the graphics canvas at a specified position.
Syntax
C#
    public void Write(string text, int x, int y, Color color,
                          string fontName = "Arial", int fontSize = 12)
Parameters
text
The text string to be drawn.
x
The X-coordinate where the text begins.
y
The Y-coordinate where the text begins.
color
The color of the text.
fontName
The name of the font to use. Defaults to Arial.
fontSize
The size of the font. Defaults to 12.
Exceptions
Exception Condition
ArgumentNullException Thrown if the text or color arguments are null.
Remarks
The font and brush objects are automatically disposed after drawing.
