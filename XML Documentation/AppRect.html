Class AppRect
Represents a rectangle drawing command for the BOOSE interpreter.
Syntax
C#
    internal class AppRect : Command
Remarks
This command does not draw directly. It only parses and stores rectangle parameters. The actual drawing is handled in the main application (e.g., Form1) after this command is executed.

Documentation Preview
Method Execute
Executes the rectangle command.
Syntax

    public override void Execute()
Remarks
The drawing logic is handled externally in the main form. This method exists to satisfy the abstract base class requirement.
