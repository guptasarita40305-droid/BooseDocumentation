Class AppPenColour
Represents the Pencolour command, which updates the drawing pen colour used for subsequent drawing operations. This class validates RGB values and creates a new Pen object.
Syntax
C#
    internal class AppPenColour : Command

Documentation Preview
Property Pen
Gets or sets the Pen object created based on the supplied RGB values. This Pen is used externally by the drawing system.

C#
    public Pen Pen

Method Execute
Executes the Pencolour command. The actual pen colour update is performed externally (e.g., in the main form), while this method exists to adhere to the Command pattern structure.

    public override void Execute()

Method CheckParameters
Validates the parameters for the Pencolour command and constructs a new Pen using the supplied RGB values.
Syntax
C#
    public override void CheckParameters(string[] parameters)
Parameters
parameters
An array containing exactly three string values representing R, G, and B colour components.
Exceptions
Exception Condition
ArgumentException Thrown when:
  • The parameter count is not exactly 3.
  • Any of the R, G, or B values are not valid integers.
