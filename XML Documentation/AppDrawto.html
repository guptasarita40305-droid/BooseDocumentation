Class AppDrawto
Represents the DrawTo command in the BOOSE drawing system. This command stores the destination coordinates (X, Y). The drawing operation is handled externally by the canvas or main form.
Syntax
C#
    internal class AppDrawto : Command

Documentation Preview
Method Execute
Executes the DrawTo command. The actual drawing is performed by the main application or canvas, so this method simply completes successfully.
