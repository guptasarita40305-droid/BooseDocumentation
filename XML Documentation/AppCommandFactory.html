Class AppCommandFactory
A custom command factory responsible for creating instances of BOOSE commands used in the assignment.
Syntax
C#
    internal class AppCommandFactory : CommandFactory
Remarks
This factory overrides CommandFactory to provide custom command mappings for Circle, MoveTo, Rectangle, and PenColour.

Method MakeCommand
Creates an instance of a command based on the provided command keyword.
Syntax
C#
    public override ICommand MakeCommand(string commandType)
Parameters
commandType
The name of the command to create.
Return Value
An instance of a class implementing ICommand corresponding to the command name, or the base factory implementation if the command is not recognized.
Remarks
Supported custom commands:
  • circle → AppCircle
  • moveto → AppMoveto
  • rect → AppRect
  • pencolour → AppPenColour
Any unrecognized command is passed to the base class for fallback creation.
