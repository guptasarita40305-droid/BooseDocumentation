Class AppCommandFactory
Provides a factory for creating BOOSE command instances.
Syntax
C#
    public class AppCommandFactory : CommandFactory
Remarks
This class implements the Factory Design Pattern by mapping command keywords to their corresponding command objects. Commands are created dynamically at runtime based on input text.
