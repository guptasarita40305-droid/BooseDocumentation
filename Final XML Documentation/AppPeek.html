Class AppPeek
Represents the peek command in the BOOSE language. This command retrieves a value from an array at a specified index and assigns it to a target variable.
Syntax
C#
    public class AppPeek : Command

Documentation Preview
Method Execute
Executes the peek command. Syntax: peek varName = arrayName index
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
ArgumentException Thrown if:
  • The parameters are fewer than 4 or incorrectly formatted.
  • The index is out of range for the specified array.
Remarks
- varName: The variable to store the retrieved value.
- arrayName: The name of the array to peek into.
- index: The position in the array to retrieve the value from.
