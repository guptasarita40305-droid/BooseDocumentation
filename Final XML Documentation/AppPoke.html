Class AppPoke
Represents the poke command in the BOOSE language. This command assigns a value to a specific index of an array.
Syntax
C#
    public class AppPoke : Command

Documentation Preview
Method Execute
Executes the poke command. Syntax: poke arrayName index = value
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
ArgumentException Thrown if:
  • The parameters are fewer than 4 or incorrectly formatted.
  • The index is out of range for the specified array.
Remarks
- arrayName: The name of the array to modify.
- index: The position in the array where the value will be stored.
- =: Assignment operator.
- value: The value to assign to the array element.
