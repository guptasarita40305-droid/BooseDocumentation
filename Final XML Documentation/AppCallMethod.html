Class AppCallMethod
Implements the BOOSE call command used to invoke user-defined methods.
Syntax
C#
    public class AppCallMethod : CanvasCommand
Remarks
The command evaluates argument expressions, assigns them to method parameters, executes the method body, and preserves the returned value while restoring the previous program state.

Supported syntax:

C#
call myMethod 10 20

Method Execute
Executes the method call command.
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
ArgumentException Thrown if the argument count does not match the method definition.
Remarks
This method performs the following steps:
  1. Retrieves the method definition from AppMethodStore.
  2. Evaluates and validates argument expressions.
  3. Saves the current values of parameters and return variables.
  4. Initialises method parameters and return variable.
  5. Executes all commands in the method body.
  6. Restores the previous variable state while preserving the return value.
