Method Execute
Executes the real command. Syntax: real varName [= expression]
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
ArgumentException Thrown if no variable name is provided in the parameters.
Remarks
- varName: The name of the variable to declare or assign.
- = expression: (Optional) An expression to evaluate and assign to the variable.
If no expression is provided, the variable defaults to 0.0.
