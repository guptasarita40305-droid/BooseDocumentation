Property class
Provides a concrete implementation of the ICanvas interface for drawing graphical shapes and text.
Syntax
C#
    public sealed class AppCanvas : ICanvas
Remarks
The canvas maintains a current pen position and colour, allowing drawing commands such as lines, circles, rectangles, and text. All drawing operations are rendered onto an internal bitmap.
