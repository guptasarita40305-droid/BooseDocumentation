Method EvaluateStringExpression
Evaluates a string expression that may contain literal strings and numeric expressions. Supports concatenation using the + operator.
Syntax
C#
    private static string EvaluateStringExpression(string expr)
Parameters
expr
The string expression to evaluate.
Return Value
A fully evaluated string where:
  • Quoted text (e.g., "Hello") is preserved as literal text.
  • Numeric expressions are evaluated and converted to string form.
  • Concatenation is handled using the + operator outside of quoted strings.
