Class AppArrayDecl
Implements the BOOSE array command, allowing declaration of integer or real arrays with a fixed length.
Syntax
C#
    public class AppArrayDecl : Command
Remarks
Supported syntax:
C#
array int numbers 10
array real values 5
The array is initialised with default values (0 for int, 0.0 for real) and stored in AppVariables.

Method Execute
Executes the array declaration command.
Syntax
C#

Remarks
Creates an array of the specified length and element type, initialises all elements to zero, and stores it in AppVariables using the provided array name.
