Class AppAssign
Implements the BOOSE assignment command, allowing values to be assigned to previously declared variables.
Syntax
C#
    public class AppAssign : Command, AppIVariableCommand
Remarks
Supported syntax:
C#
x = 10
y = x + 5
The command evaluates the expression based on the existing variable type (int or real).
